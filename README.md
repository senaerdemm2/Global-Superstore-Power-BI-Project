# Global-Superstore-Power-BI-Project
## Global Superstore Power BI Dashboard
This project visualizes and analyzes the Global Superstore dataset using Power BI.
It includes insights into sales performance, customer behavior, product categories, and regional trends.
    File Name                      Description                                              
  
 `Overview Page.png`               General overview of key performance indicators (KPIs).   
 `Product Base Analysis Page.png`  Analysis of product categories and sub-categories.       
 `Customer Based Analysis.png`     Insights into customer segmentation and profitability.   
 `Region Based Analysis.png`       Regional sales and profit distribution.                  
 `Tooltip.png`                     Example of tooltip functionality used in visualizations. 
 `Drillthrough Page.png`           Drillthrough details for deeper data exploration.        
 `global_superstore.xlsx`          Source dataset used in the Power BI dashboard.    


## Tools Used
Power BI Desktop
Microsoft Excel

## How to Use

Download or clone this repository to your local machine.
Open Power BI Desktop.
Load the dataset global_superstore.xlsx by selecting
Home → Get Data → Excel Workbook → Select the file.
Use the uploaded screenshots (.png files) as a reference to recreate the visuals and page layouts in Power BI.
Each image represents one report page (e.g., Overview, Product Analysis, Customer Analysis, etc.).
Customize visuals, measures, and formatting based on your preferences or analysis goals.
